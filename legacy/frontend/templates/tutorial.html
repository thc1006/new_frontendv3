Create Page
Project Name:

You can specify a name for your new project.

In the "Project Name" field, type the desired name for your project.

This name will be used to identify your project in the website.

3D Floor Plan:

You have two options to add a 3D model with the "Upload" switch.

  • Select Existing Model:

    You can choose from existing models that you have access to.

    If you prefer to use an existing model, ensure the "Upload" switch is off.

    Choose a model from the dropdown list of available models.

  • Model Upload:

    Toggle the switch next to "Upload" to enable file upload.

    Click the file input field that appears and select a .gltf file from your device.

Mapbox:

This website uses Mapbox to render maps and allows precise placement of the 3D model based on longitude and latitude coordinates.

The map will load with your selected model.

You can recenter, drag, and rotate the selected model on the map to fit your desired location. *Only when you have uploaded a file.

Use the following controls to adjust the model:

  • Recenter: Click the "Recenter" button to move the model to the center of the map.
  • Drag the Model: Click on the model while holding the Shift key to move it around the map.
  • Rotate the Model: Hold the Alt key and use the left or right arrow keys to rotate the model in smaller steps.
Create:

After setting up the model, you can create the project or cancel the process.

Once satisfied with your setup, click the "Create" button to finalize and create the project.

If you decide not to proceed, click "Cancel" to return to the project overview without saving any changes.
